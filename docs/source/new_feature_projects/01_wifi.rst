Wi-Fi
========================================

Arduino UNO R4 WiFiは、内蔵されたESP32-S3モジュールを搭載しており、Wi-Fi®ネットワークに接続し、ネットワーク操作を行うことができます。HTTPS、MQTT、UDPなどのプロトコルをサポートしており、これらはテスト済みで互換性があります。
