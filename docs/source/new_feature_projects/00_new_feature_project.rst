Arduino® UNO R4 WiFiの探求
========================================

.. image:: img/unor4wifi.webp
    :align: center
    :width: 50%

**Arduino Uno R4 WiFi board** はArduinoファミリーの最新の追加で、高度な機能を提供しながら、あなたが期待する堅牢性と汎用性を維持しています。このボードは、32ビットのArm® Cortex®-M4マイクロコントローラーによる大幅な処理能力の向上をもたらし、UNOシリーズの標準的なフォームファクターとシールドの積み重ね性との互換性を保持しています。

**Uno R4がR3と比べて新しい点は？**

- **強化された処理能力**：8ビットのAVRから32ビットのArm® Cortex®-M4マイクロコントローラーに移行。
- **メモリアップグレード**：32KBのSRAMと256KBのNANDフラッシュメモリでプロジェクトを向上させます。
- **高度な接続性**：USB-CとWi-Fiでシームレスな接続を実現。
- **向上したパフォーマンス**：より速い処理と大容量メモリでより多くのことを実現。
- **柔軟な電源供給**：最大24Vでボードに電力を供給し、プロジェクトの可能性を広げます。

上記のアップグレードに加えて、R4 WiFiは以下の新機能を導入しています：

**新機能**

- **Wi-Fi®**：ESP32-S3モジュールによって可能にされた、さまざまなIoTプロジェクトに理想的なワイヤレス接続性を提供します。
- **Bluetooth®**：デバイス間の短距離ワイヤレス通信を提供し、ESP32-S3モジュールによって駆動されます。
- **内蔵リアルタイムクロック（RTC）**：時間に敏感なアプリケーションに最適です。バッテリー駆動とボードをオフにしながらRTCを稼働させるための「OFF」ピンに追加のピンが含まれています。
- **12x8 LED Matrix**：データを表示したりアニメーションを作成するための簡単な方法。
- **DACチャネル**：オーディオプロジェクトに最適な、正確なアナログ出力を実現します。
- **HIDサポート**：内蔵のHIDサポートを使用して、USB経由でマウスやキーボードをシミュレートします。
- **CANプロトコルサポート**：自動車や産業アプリケーションへのアプローチを拡張します。
